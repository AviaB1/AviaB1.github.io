# AviaB's Blog

A personal cybersecurity blog built with [Astro](https://astro.build/), [Tailwind CSS](https://tailwindcss.com/), and [shadcn/ui](https://ui.shadcn.com/).

## About

This blog covers topics in malware analysis, DFIR (Digital Forensics and Incident Response), detection engineering, and reverse engineering.

**Live site:** [https://AviaB1.github.io](https://AviaB1.github.io)

## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/AviaB1/AviaB1.github.io.git
   cd AviaB1.github.io
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open your browser and visit `http://localhost:1234`.

## Available Commands

| Command           | Description                                                     |
| ----------------- | --------------------------------------------------------------- |
| `npm run dev`     | Start the development server                                    |
| `npm run build`   | Run type checking and build the project                         |
| `npm run preview` | Preview the built project                                       |
| `npm run prettier`| Format all files using [Prettier](https://prettier.io/)         |
